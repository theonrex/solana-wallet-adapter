# BaseWalletConnectButton

## **Overview**  
`BaseWalletConnectButton` is a UI component from `@solana/wallet-adapter-react-ui` that provides a button for users to connect their Solana wallet. It handles state management, UI updates, and interaction with `useWalletConnectButton` internally.  

## **Import**  
To use `BaseWalletConnectButton`, import it as follows:  
```tsx
import { BaseWalletConnectButton } from '@solana/wallet-adapter-react-ui';
```

## **Usage**  
You can use `BaseWalletConnectButton` inside your React components to allow users to connect their wallets.  
```tsx
import React from 'react';
import { BaseWalletConnectButton } from '@solana/wallet-adapter-react-ui';

export default function ConnectWallet() {
    return <BaseWalletConnectButton labels={{ connecting: 'Connecting...', ready: 'Connect Wallet' }} />;
}
```

## **Props (Parameters)**  
`BaseWalletConnectButton` accepts the following props:  

| **Prop**   | **Type**  | **Description** |
|------------|-----------|----------------|
| `labels`  | `{ [TButtonState: string]: string }` | A mapping of button states (`connecting`, `ready`, etc.) to display labels. |
| `disabled` | `boolean` | (Optional) If `true`, disables the button. |
| `onClick`  | `(event: React.MouseEvent<HTMLButtonElement>) => void` | (Optional) Callback function triggered when the button is clicked. |

## **Return Type**  
`BaseWalletConnectButton` returns a React component that renders a styled connect button.  

## **Example with Custom Labels**  
```tsx
import React from 'react';
import { BaseWalletConnectButton } from '@solana/wallet-adapter-react-ui';

export default function ConnectButtonExample() {
    return (
        <BaseWalletConnectButton
            labels={{
                connecting: 'Connecting...',
                ready: 'Sign In',
            }}
        />
    );
}
```
