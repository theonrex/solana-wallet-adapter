# WalletModal

## Overview
The `WalletModal` component from `@solana/wallet-adapter-react-ui` provides a user-friendly interface for selecting and connecting Solana wallets. It lists available wallets and allows users to connect with a single click.

## Import
To use the `WalletModal` component, import it as follows:

```tsx
import { WalletModal } from '@solana/wallet-adapter-react-ui';
```

## Usage
The `WalletModal` is typically used within a Solana application to provide a UI for users to select and connect their wallets.

```tsx
import { WalletModal } from '@solana/wallet-adapter-react-ui';

const MyComponent = () => {
    return (
        <div>
            <WalletModal />
        </div>
    );
};
```
