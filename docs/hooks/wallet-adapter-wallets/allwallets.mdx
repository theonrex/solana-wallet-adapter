# Solana Wallets Integration Guide

This guide provides an overview of various Solana wallets, their integration into your frontend application, and the necessary imports to use them effectively.

## 1. Overview
Solana supports multiple wallets that allow users to interact with dApps securely. The `@solana/wallet-adapter` ecosystem provides a unified way to integrate these wallets into your application.

## 2. Installing Required Dependencies
Before integrating wallets, install the required packages:
```sh
npm install @solana/wallet-adapter-wallets @solana/wallet-adapter-react-ui @solana/wallet-adapter-base
```

## 3. Importing Wallet Adapters
To use the wallets in your frontend, import the necessary adapters:
```ts
import {
    PhantomWalletAdapter,
    SolflareWalletAdapter,
    LedgerWalletAdapter,
    TorusWalletAdapter,
    MathWalletAdapter,
    TrustWalletAdapter,
    Coin98WalletAdapter,
    WalletConnectWalletAdapter,
} from "@solana/wallet-adapter-wallets";
```

## 5. Using Wallets in Your Application
After importing the wallets, configure them in your Solana dApp as follows:
```ts
import { WalletProvider } from '@solana/wallet-adapter-react';
import { ConnectionProvider } from '@solana/wallet-adapter-react';

const wallets = [
    new PhantomWalletAdapter(),
    new SolflareWalletAdapter(),
    new LedgerWalletAdapter(),
    new TorusWalletAdapter(),
];

function App() {
    return (
        <ConnectionProvider endpoint={"https://api.mainnet-beta.solana.com"}>
            <WalletProvider wallets={wallets} autoConnect>
                <YourComponent />
            </WalletProvider>
        </ConnectionProvider>
    );
}
```
