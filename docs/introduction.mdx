# Introduction

The Solana Wallet Adapter is a powerful library that simplifies the integration of Solana wallets into your decentralized applications (dApps). It provides a unified interface for interacting with various Solana wallet providers, making it easier for developers to support multiple wallets without having to implement separate integrations for each one.

## Key Features

- **Unified Wallet Interface**: Interact with multiple Solana wallets using a single, consistent API.
- **React Hooks**: Easily access wallet states and functions in your React components.
- **Auto-connect**: Automatically reconnect to the last used wallet for a seamless user experience.
- **Extensible**: Support for adding new wallet adapters as they become available.
- **TypeScript Support**: Full TypeScript support for improved developer experience and type safety.

## Why Use Solana Wallet Adapter?

1. **Simplicity**: Reduce the complexity of integrating multiple wallet providers.
2. **Consistency**: Provide a consistent user experience across different wallets.
3. **Flexibility**: Easily add or remove support for different wallet providers.
4. **Community Support**: Benefit from a widely-used and actively maintained library in the Solana ecosystem.

## Supported Wallets

The Solana Wallet Adapter supports a wide range of popular Solana wallets, including but not limited to:

- Phantom
- Solflare
- Ledger
- Slope
- Sollet
- MathWallet

For a complete list of supported wallets and their respective adapters, please refer to the [API Reference](./hooks/wallet-adapter-wallets/allwallets.mdx) section.
